map_hint() <= 맵의 중심 포인트 입력

절대 좌표 반환(w,h) = cvtpointer() <= 센싱을 통해 측정된 거리와 측정하였던 각도

상대 좌표 반환 re_point() <= 절대좌표를 상대좌표로 변환

h_input() <= 상대좌표에 맞는 장소에 선을 그림

run() <= 센서를 부착한 물체가 음직이면 움직이는 거리와 각도를 입력
    각도는 상대각도로 입력
    ex)
        2초 동안 90 이동시
        run(0,45)
        (1초 딜레이)
        run(0,90)
        이 아닌
        run(0,45)
        (1초 딜레이)
        run(0,45)
        로 입력
